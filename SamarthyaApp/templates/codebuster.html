About the event

You'll be given some code for this event and need to figure out where the mistakes are. The code will be provided in C language.
To achieve the intended result, errors must be found.

Rules and Regulations

  1. Points will be given according to how complicated the error was.
  2. Incorrect answers will result in points being taken.
  3. The winner will be decided on Level 2.
  4. The winner is the person with the most points, followed by the runner.
